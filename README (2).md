## Featured Projects

**HealthBot**
A chatbot that answers general health-related questions using a language model, built with Python and Streamlit. Built mainly to learn how to connect a UI to an API and handle conversational state.
`Python` `Streamlit`

**ChatPDF**
A tool that lets a user upload a PDF and ask questions about its content. Handles text extraction and retrieval so answers are based on the actual document instead of general knowledge.
`Python` `Streamlit`

**Scheme Recommendation System**
A simple recommendation tool that suggests government welfare schemes to users based on inputs like income, occupation, and category. Built to practice basic rule-based logic and form-driven UIs.
`Python` `Streamlit`
